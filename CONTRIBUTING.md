# Contributing to Toolz

Toolz ships one tool per week. Each tool lives independently under `apps/web/src/tools/<tool>`
with its own storage, types, and README — please keep changes to one tool scoped to that folder.

1. Fork and branch from `main`.
2. Run `npm install && npm run dev` inside `apps/web`.
3. Keep tools local-first: no required login, no required backend.
4. Open a PR referencing the relevant `tool:<name>` label.
